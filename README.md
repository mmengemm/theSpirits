# UBS Login script

Script to easy your morning pain

## Requirements

[Python v3.6+](https://www.python.org/downloads/)

pip Modules: [PyAutoGUI](https://pypi.org/project/PyAutoGUI/), [opencv-python](https://pypi.org/project/opencv-python/), [playsound](https://pypi.org/project/playsound/) (Will be installed automatically by the script)

## Running the script

```shell
python project.py
```
Or double click on `project.py`

**Once you here the sound you need to scan the QR code manually**

*The scripts creates a folder `.autoUBS` in your home directory with the required files*
